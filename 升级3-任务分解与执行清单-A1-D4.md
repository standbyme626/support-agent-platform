# 升级3任务分解与执行清单（A1-D4）

> 版本目标：升级3“文件级改动 + refs参考 + 测试验收 + 量化阈值”执行版
> 主规范来源：[`升级3.md`](./升级3.md)、[`升级3-实施要求与验收测试规范.md`](./升级3-实施要求与验收测试规范.md)

## 0. 使用方式（必须）

1. 按 `A1 -> A2 -> A3 -> A4` 的顺序执行每组任务，禁止跳过 `x4` 验收任务。
2. 每项完成后输出：改动文件、测试结果、量化结果、风险点。
3. 阶段切换前必须满足上一阶段退出标准。
4. 全局质量门禁必须通过：`make validate-structure`、`make lint`、`make typecheck`、`make test`、`make check`、`make acceptance`。
